# Thinking OS / MindOS

MVP frontend personal thinking system on Next.js.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui-style components

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment

Create `.env.local` from `.env.example`.
